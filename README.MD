## Install

```
yarn
```

---
## Simple Use Case

```js

yarn start

```
It will start a server listening in port 80 which is configurable.
After receiving a request from client successfully to http://localhost/health
the server then responses with an object:

```js
{ result: 'OK' , timestamps: Date.now()}
```
